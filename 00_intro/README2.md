### 코드블럭(code block)

 1. 코드박스

    - "`"(백킷) 세개로 입력 후 엔터!
    - 내가 원하는 언어로 설정이 가능하다.

    ```python
    class = '부울경 2반'
    ```

    

 2. 인라인코드

    - "`"(백킷) 안에 코드/문서 내용을 작성해준다.
    - `printf('부울경 2반')` -> `print('부울경 2반')`

### 텍스트 강조(Text Emphasis)

- 굵게:
  - `*`를 텍스트 앞뒤에 2개씩 씌어준다.
  - **부울경**
- 기울임
  - `*`를 텍스트 앞뒤에 1개씩 씌어준다.
  - *부울경*
- 취소선
  - `~`를 텍스트 앞뒤에 2개씩 씌어준다.
  - ~~구미~~

​	***싸피7기***
